# Leftovers Bot

### Enviroment: With pip vertial environment

python request: `3.9.6`

```
pip3 install virtualenv
virtualenv venv --python=python3.9.6
source venv/bin/activate
pip install -r requirements.txt
deactivate
rm -rf venv     # remove the venv
```
